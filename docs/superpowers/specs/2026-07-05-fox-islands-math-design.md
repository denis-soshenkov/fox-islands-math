# Лисьи острова — весёлая математика (дизайн)

Браузерный аналог Funexpected Math: обучающая математическая игра для детей 3–7 лет.
Автономный режим: пользователь недоступен для вопросов, решения зафиксированы из явного
запроса («любые технологии, любые ассеты, главное — чтобы работала как оригинал»).

## Ограничение по правам

Оригинальные ассеты, персонажи и название Funexpected — защищённая собственность.
Клонируются **механики и структура** (не защищаются авторским правом), графика —
собственная: emoji, SVG, CSS. Название своё: «Лисьи острова».

## Целевой опыт (как в оригинале)

- Карта мира с тематическими островами, по ним — активности.
- Маскот-проводник (лисёнок Фокси) с репликами и голосовыми заданиями (Web Speech API, ru-RU).
- Короткие мини-игры: 3–5 раундов, задания генерируются процедурно.
- Управление только тапами — без чтения, без drag-and-drop (надёжно на телефоне и ПК).
- Награды: 1–3 звезды за сессию (по числу ошибок), уровни 2 и 3 открываются последовательно.
- Прогресс в localStorage. Раздел для родителей за «удержи кнопку» — таблица прогресса, сброс.
- Звук: синтезированные эффекты WebAudio (без файлов), выключатель звука.

## Технологии

Vanilla HTML/CSS/JS, без сборки и зависимостей. Обычные `<script>` (не модули), чтобы
работало и с `file://`, и с любым статическим сервером. Namespace `window.FX`.

## Острова и 12 мини-игр (3 уровня сложности в каждой)

| Остров | Активности |
|---|---|
| Остров Счёта | counting «Весёлый счёт» (посчитай и выбери цифру), digits «Цифры» (цифра → тарелка с таким числом предметов) |
| Остров Сравнений | compare «Больше — меньше», sorting «Сортировка» (разложи по корзинам) |
| Остров Фигур | shapes «Найди фигуры», symmetry «Зеркало» (дострой симметричный узор на сетке) |
| Остров Узоров | patterns «Продолжи узор», memory «Найди пару» (цифра ↔ количество) |
| Остров Арифметики | addition «Сложение» (наглядные суммы, L3 — пропущенное слагаемое), subtraction «Вычитание» (предметы улетают) |
| Остров Приключений | dots «Точки по порядку» (соедини 1…N — появится картинка), missing «Пропущенное число» (числовой ряд, L3 — обратный счёт и счёт двойками) |

## Архитектура

- `index.html` — оболочка, порядок скриптов: utils → audio → data → games/* → engine.
- `js/utils.js` — helpers (rand, shuffle, el, delay) + реестр `FX.register(game)`.
- `js/audio.js` — WebAudio-синтез (correct/wrong/pop/fanfare/click), speech(ru), mute.
- `js/data.js` — конфиг островов, наборы emoji, палитры, фразы похвалы.
- `js/engine.js` — экраны (дом, карта, остров, игра, результаты, родители), раннер
  раундов, подсчёт ошибок и звёзд, прогресс в localStorage, конфетти, пузырь лисёнка.
- `js/games/*.js` — 12 файлов; контракт: `{id, name, icon, skill, rounds?, newRound(level, api)}`.
  API движка: `stage, level, prompt(), correct(), wrong(), complete(), reportMistakes(n), sfx(), say()`.

Звёзды: 0 ошибок → 3★, 1–2 → 2★, больше → 1★. Уровень k+1 открыт при ★ ≥ 1 на уровне k.

## Проверка

Запуск статического сервера + прогон в браузере (Claude Preview): загрузка без ошибок в
консоли, прохождение нескольких игр до экрана звёзд, сохранение прогресса после reload,
мобильный вьюпорт.
